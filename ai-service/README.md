# 🚀 Stock Price Prediction Service (LSTM)

บริการวิเคราะห์แนวโน้มราคาหุ้นรายวันโดยใช้ Deep Learning (LSTM Model) พร้อมระบบรองรับ GPU (CUDA) ในตัว

---

## 🧐 คำอธิบายค่าสถิติ (Metrics) ในการเทรน

ในขณะที่คุณเทรนโมเดล คุณจะเห็นค่าหลักๆ 4 ตัว คือ:

| ค่าคงที่ | ความหมาย | เป้าหมาย |
| :--- | :--- | :--- |
| **Loss** | ความผิดพลาดจากข้อมูลที่กำลังเทรน (Train Data) | **ยิ่งน้อยยิ่งดี** |
| **Accuracy** | ความแม่นยำจากข้อมูลที่กำลังเทรน | **ยิ่งมากยิ่งดี** |
| **val_loss** | ความผิดพลาดจากข้อมูลที่โมเดลไม่เคยเห็น (Validation Data) | **ต้องน้อย** (หากค่านี้น้อยแต่ Loss ต่ำมาก แสดงว่าโมเดลท่องจำ) |
| **val_accuracy** | ความแม่นยำจริงเมื่อโมเดลเจอข้อมูลใหม่ๆ | **นี่คือตัวเลขที่เชื่อถือได้ที่สุด** |

---

## 🏗️ โครงสร้างการทำงานภายใน

ระบบนี้ทำงานเป็นขั้นตอน (Pipeline) แบบมาตรฐานสากล:

### 1. Data Processing (การเตรียมข้อมูล)
- **Fetching**: ดึงข้อมูลราคาหุ้น (OHLCV) ย้อนหลังหลายปีจาก Yahoo Finance ผ่าน `yfinance`
- **Features Engineering**: สร้างดัชนีทางเทคนิคเพิ่ม (RSI, EMA_50) เพื่อให้ AI เข้าใจสภาพตลาดมากขึ้น
- **Scaling**: ปรับตัวเลขให้อยู่ในระดับ 0 ถึง 1 (MinMaxScaler) เพื่อให้ AI ประมวลผลได้เสถียรขึ้น

### 2. LSTM Architecture (โครงสร้าง AI)
เราใช้ **Long Short-Term Memory (LSTM)** ซึ่งเป็น Neural Network ชนิดพิเศษที่เก่งเรื่องข้อมูลที่มี "ลำดับเวลา" (Time Series) เพราะมันสามารถจำแนวโน้มในอดีตมาช่วยตัดสินใจในปัจจุบันได้

### 3. Training Logic
- **Sliding Window**: มัดรวมข้อมูลย้อนหลัง 60 วัน เพื่อทำนายผลลัพธ์ในวันที่ 61
- **Temporal Split**: แบ่งข้อมูลเป็น 80% สำหรับเทรน และ 20% สำหรับทดสอบ (โดยยึดตามลำดับเวลา)
- **Callbacks**: 
    - `EarlyStopping`: หยุดเทรนเองหากโมเดลไม่พัฒนาแล้ว
    - `ModelCheckpoint`: บันทึกเฉพาะโมเดลเวอร์ชันที่แม่นยำที่สุดเท่านั้น

## 🛠️ วิธีการใช้งาน (Usage Guide)

### 1. การเทรนโมเดล (Training)
คุณสามารถสั่งเทรนหุ้นที่ต้องการได้ผ่าน Terminal:

```powershell
# เทรนหุ้นตั้งต้น (NVDA)
python train_model.py

# เทรนหุ้นอื่นๆ และกำหนดรอบการเทรนเอง
python train_model.py --ticker AAPL --epochs 100 --batch_size 64
```

### 2. การทำนายผล (Prediction)
ใช้โมเดลที่บันทึกไว้มาวิเคราะห์ทิศทางในวันพรุ่งนี้:

```powershell
# วิเคราะห์หุ้น NVDA
python main.py

# วิเคราะห์หุ้นที่คุณเพิ่งเทรนไป
python main.py --ticker AAPL
```

## 📂 โครงสร้างโฟลเดอร์ (File Directory)
- `src/config.py`: ศูนย์รวมการตั้งค่าทั้งหมด (Hyperparameters)
- `src/model.py`: โครงสร้าง Neural Network (LSTM)
- `src/data_pipeline.py`: ระบบดึงและทำความสะอาดข้อมูล
- `saved_models/`: ที่เก็บไฟล์โมเดล (.keras) และตัวปรับสเกล (.pkl)
- `cuda_setup.py`: ไฟล์ช่วยเปิดใช้งาน GPU สำหรับ Windows Native

---
💡 **Tips**: หากค่า `val_loss` เริ่มสูงขึ้นเรื่อยๆ ขณะที่ `loss` ลดลง แสดงว่าโมเดลเริ่ม Overfitting (เริ่มจำข้อสอบ) ระบบ EarlyStopping จะทำหน้าที่ตัดจบการเทรนให้คุณโดยอัตโนมัติ
